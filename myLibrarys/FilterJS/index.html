Функция фильтрации контента
FilterJS

Первое, что нам нужно — это контент, затем нужно сделать панель. В качестве контента мы будем подразумевать какие-либо логические блоки: карточки товаров, фотографии (с каким-то описанием),

Работа функции не должна зависеть от типа контента. Тип, конечно, влияет на набор применяемых инструментов, но JavaScript внутри телы основной программы не должен разбираться с семантикой конечного пользователя.

Также очень важное замечание: функция НЕ занимается сортировкой — только фильтрация. То есть она либо пропускает, либо не пропускает контент - не более того.

Основные понятия

  1. Список - набор вариантов одной характеристики, по которой происходит фильтрация.
  2. Данные - набор однотипных списков характеристик различных объектов.
Режимы работы фильтров
  1. Активное состояние - Основной режим, в котором фильтр отсекает ненужный контент
  2. Фильтр сброшен - Режим, в котором фильтр отключен пользователем.
  3. Фильтр заблокирован - В соответствии с внутренней логикой сайта, определённые комбинации логически не могут пересекаться, поэтому разработчик может установить этот режим.

Виды фильтров

Логически можно определить несколько видов фильтров

  1. от 0 до 1 из 2 На панели представляется в виде двух элементов, активное состояние которых никогда не совпадает, но одновременно они могут быть сброшены.

    (или или, либо ничего)

    Например: мясо или курица.
  2. от 0 до 1 из бесконечности По большей части — это режим работы выбора одного элемента из списка.

    (один, либо ничего)

    Например: артикул компьютерного кресла.
  3. от 0 до бесконечности из бесконечности Множественный выбор из списка.

    (несколько на выбор)

    Например: дополнительные опции к заказу.
  4. 1 из бесконечности Подразумевается, что фильтр в любом случае будет установлен: либо по умолчанию, либо в соотвествии с выбором пользователя.

    (один на выбор)

    Например: тариф эконом выбирается по умолчанию, можно выбрать комфорт, люкс и т. д.
  5. диапазон Выбор из упорядоченного набора данных: выбор нижней границы и верхней.

    (от и до)

    Например: минимальная цена и максимальная (ползунок).
